# Emotion-Driven AI Story & Visual Generator

A Python project that combines *AI-powered story generation* with *abstract visual creation*.  
The system analyzes text to detect emotion and generates *a story continuation* and a *corresponding abstract image or animated GIF* that reflects the emotional tone.

---

## Features

- Generate a story continuation from a short text prompt using GPT-2
- Detect emotions: happy, sad, neutral, mysterious, angry
- Create abstract visuals representing the emotion:
  - Colors, shapes, patterns
  - Optional animated GIF output
- Modular and interactive Python code
- Ready for CLI or web interface (Streamlit/Gradio)

---

## Demo

Example prompt:
In the abandoned castle, shadows moved as if alive.
-> 
Detected Emotion: **Mysterious**  

Generated Story:In the abandoned castle, shadows moved as if alive, whispering secrets that only the brave could hear. The moonlight cast eerie patterns, guiding the explorer deeper into the unknown corridors...
